## 4. OpenClaw 多 Agent + 多 Bot 配置研究

Eyjan 询问如何配置多个 Agent（不同模型）+ 多个 Discord Bot

**研究成果**：

### 单 Gateway + 多 Agent（推荐方式）
```json5
{
  "agents": {
    "list": [
      { "id": "ego", "model": { "primary": "moonshot/kimi-k2.5" } },
      { "id": "code-helper", "model": { "primary": "anthropic/claude-opus-4-6" } }
    ]
  },
  "channels": {
    "discord": {
      "accounts": {
        "main": { "token": "xxx" },
        "code": { "token": "yyy" }
      }
    }
  },
  "bindings": [
    { "agentId": "ego", "match": { "channel": "discord", "accountId": "main" } },
    { "agentId": "code-helper", "match": { "channel": "discord", "accountId": "code" } }
  ]
}
```

**关键配置项**：
- `agents.list[]` - 定义多个 Agent，每个可有不同模型、workspace
- `channels.discord.accounts` - 同一渠道多账号（多 Bot）
- `bindings[]` - 路由规则，将消息匹配到指定 Agent

**协作方式**：
- `sessions_send()` - Agent 间直接发消息
- `sessions_spawn()` - 生成子代理处理任务
- 共享 workspace 文件

**文档来源**：
- `/opt/openclaw/docs/concepts/multi-agent.md`
- `/opt/openclaw/docs/channels/discord.md`
- `/opt/openclaw/docs/gateway/configuration.md`
