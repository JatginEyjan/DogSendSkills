## 核心设计确认

### 游戏循环
```
完成订单 → 获得经验 → 掌门升级 → 解锁新内容 → 弟子变强 → 更多/更难订单
```

### 关键系统
1. **订单系统**: 5槽位, 20秒自动刷新, 手动刷新(5次/天, 100灵石)
2. **掌门境界**: 完成X个订单升级, X递增(3→5→8→12→20)
3. **弟子系统**: 蓝/紫/橙品质, 消耗灵石升级, 有名字和背景
4. **战场系统**: 自动战斗, 弟子死亡暂停恢复
5. **建筑系统**: 仙栈(槽位), 药坊(恢复), 武器/防具店(预留)

### 体验重点
- **升级仪式感**: 金光特效 + 弟子祝贺 + 解锁预览
- **资源缺口**: 灵石始终不够, 需要决策分配
- **弟子身份**: 随机名字 + 背景故事
